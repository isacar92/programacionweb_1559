Formas de agregar estilos a nuestra página web

Estilos en línea

Estos estilos en línea se agregan directamente en el elemento html, mediante el atributo style

Este parrafo tiene estilo :3

Estilos internos

Estos estilos se agregan en el mismo archivo HTML, dentro de la etiqueta <head> y se utiliza la etiqueta <style>

Estilos Externos

Estos estilos son los que se recomiendan, ya que tienen modularidad, facil mantenimiento, orden, eficiencia entre otras ventajas, para agregar una hoja de estilos externa, dentro de la etiqueta <head>, se va a utilizar la etiqueta <link>

Diferencias entre clases e id en CSS

Clases: Se ocupan cuando hay que reutilizar estilos que se repiten a lo largo de la página


ID: Se ocupan cuando vamos a aplicar estilos especificos a elementos o secciones única en nuestra página, y para navegar a secciones de nuestra misma página
